# whale

## Data & tasks

Tasks/data live in SQLite — query the DBs directly, no API needed. See **[docs/data-locations.md](docs/data-locations.md)** for tables and ready-to-run queries.

Quick map:
- **whale** `data/whale.db` → `proposed_tasks` (backlog) + `inbox_entries` (store dumps / raw captures).
- **krill** `../krill/data/tasks.db` → `tasks`, `projects` (active DB; set by `DB_PATH` in `krill/.env.local`).
- Not SQLite: `data/context/*.md` (per-project CONTEXT, incl. distilled `## Decisions` / `## Standing principles`) and `data/usage.jsonl` (per-call LLM metering; `GET /api/usage`).
- `*.db` are gitignored. `test.db` files are test fixtures (regenerated by `npm test`), not noise.
